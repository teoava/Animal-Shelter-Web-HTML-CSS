# Animal Shelter Web Interface (HTML5 & CSS3)

O platformă web modernă și responsivă dedicată unei asociații de protecție a animalelor, creată pentru a demonstra abilități de Front-End Development și UI/UX Design.

## 🛠️ Tehnologii și Limbaje
- **HTML5:** Structurare semantică, formulare de contact, integrare media (video/foto) și tabele de date.
- **CSS3:** - Layout-uri flexibile folosind **Flexbox**.
  - Design responsiv prin **Media Queries** (adaptare pentru mobil/tabletă).
  - Efecte vizuale moderne: **Transitions**, **Shadows** și **Gradients**.
- **Font-Awesome:** Integrare de librării externe pentru iconițe Social Media.

## 🚀 Funcționalități Cheie
- **Interfață Adaptabilă:** Site-ul se recalibrează automat în funcție de dimensiunea ecranului.
- **Navigație "Sticky":** Meniul principal rămâne fixat în partea de sus pentru o navigare ușoară.
- **Experiență Utilizator (UX):** Efecte de zoom la trecerea mouse-ului peste galeria de animale și butoane interactive.
- **Sistem de Contact:** Formular complet pentru voluntari și link-uri active către platforme de donații.

## 🎓 Context Academic
Proiect realizat în cadrul facultății **ETTI - Universitatea Transilvania din Brașov**, cu scopul de a exemplifica modul în care tehnologiile web pot susține cauze sociale prin design intuitiv și funcționalitate.
